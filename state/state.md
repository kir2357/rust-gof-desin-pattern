# Stateパターン

状態をクラスとして表現する

## 利点

通常、条件(状態)に一致するか否かの処理は、if文を使用し、コーディングする

条件分岐数が多い場合、メンテナンスし辛い

状態状態を個々のクラスで表現するため、単純明快になる
